# Android系统



|            |            | 说明                                                    | 是否可卸载 | 是否需要校验系统签名 |
| ---------- | ---------- | ------------------------------------------------------- | ---------- | -------------------- |
| 内置应用   | vendor     | 开机被安装，获取权限要验证签名                          |            |                      |
|            | system     | root级别，不校验签名就可以获取权限                      | ×          | ×                    |
| 非内置应用 | 系统签名   | 加android:sharedUserId="android.uid.system"不用校验权限 |            |                      |
|            | 非系统签名 | 普通应用，需要校验权限                                  |            |                      |
